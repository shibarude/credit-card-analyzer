# クレジットカード支出分析ツール

クレジットカード明細のCSVファイルを読み込んで、カテゴリ別・月別の支出分析とグラフ化を行うPythonツールです。

## 機能

- 📊 カテゴリ別・月別の支出集計
- 📈 複数のグラフ生成：
  - 月次支出推移（折れ線グラフ）
  - カテゴリ別月次支出（積み上げ棒グラフ）
  - カテゴリ別構成比（円グラフ）
  - カテゴリ×月ヒートマップ
- 📋 集計結果をCSVで出力
- 🏷️ 自動カテゴリ判定（商品名から）

## インストール

### 必要な環境
- Python 3.7以上

### ライブラリのインストール

```bash
pip install -r requirements.txt
```

## 使い方

### 1. CSVファイルの準備

カード明細CSVファイルは以下の形式である必要があります：

```
カード明細,支払日,請求額,,,タイトル
,,,,,利用日,利用店名及び商品名,...,金額
日付,,,,,2024-01-15,スーパー太郎,.....,5000
```

**重要な列：**
- A列（7列目）: 利用日（YYYY-MM-DD形式）
- B列（8列目）: 利用店名及び商品名
- F列（12列目）: 金額

### 2. スクリプト実行

```bash
python analyze_spending.py -i your_card_statement.csv -o output_folder
```

**オプション：**
- `-i, --input`: 入力CSVファイルパス（必須）
- `-o, --output`: 出力フォルダ（デフォルト: `output`）
- `-y, --year`: 分析対象年（デフォルト: CSVデータから自動判定）

### 3. 出力ファイル

実行後、`output`フォルダに以下が生成されます：

- `monthly_spending.csv` - 月別・カテゴリ別支出集計
- `category_summary.csv` - カテゴリ別支出サマリー
- `monthly_trend.png` - 月次支出推移グラフ
- `category_stacked.png` - カテゴリ別月次支出グラフ
- `category_pie.png` - カテゴリ別構成比（円グラフ）
- `heatmap.png` - カテゴリ×月ヒートマップ

## カテゴリ判定ルール

商品名から自動的にカテゴリを判定します：

- **食費**: スーパー、コンビニ、飲食店など
- **交通**: ガソリン、電車、タクシーなど
- **医療**: 薬局、病院など
- **通信**: 携帯、インターネットなど
- **娯楽**: 映画、ゲーム、コンサートなど
- **その他**: 上記に該当しないもの

カテゴリを自定義したい場合は、`config.yaml`を編集してください。

## 例

```bash
python analyze_spending.py -i card_2024.csv -o results_2024
```
